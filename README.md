# Veranstaltungen Bremen

Eine einfache Web-App, die alle aktuellen Veranstaltungen in Bremen übersichtlich anzeigt.

## Features

- **18 Events** aus verschiedenen Kategorien (Musik, Theater, Sport, Kultur, Festival, Kinder, Food & Drink, Comedy)
- **Kategoriefilter** – mit einem Klick nach Thema filtern
- **Volltextsuche** – nach Titel, Ort, Beschreibung und Tags
- **Sortierung** – nach Datum, Name oder Preis
- **Detail-Modal** – Klick auf ein Event zeigt alle Informationen inkl. Adresse
- **Responsive Design** – optimiert für Handy und Desktop
- **Keine Abhängigkeiten** – eine einzige HTML-Datei, läuft direkt im Browser

## Vorschau

Veranstaltungsorte wie das Weserstadion, Die Glocke, Schlachte, Theater Bremen, Übersee-Museum und viele mehr.

## Verwendung

Die App besteht aus einer einzelnen Datei `index.html` und benötigt keinen Server oder Build-Prozess.

```bash
git clone https://github.com/bc24/veranstaltungen-Bremen.git
cd veranstaltungen-Bremen
# index.html im Browser öffnen
```

Oder direkt online aufrufen: **[bc24.github.io/veranstaltungen-Bremen](https://bc24.github.io/veranstaltungen-Bremen)**

## Technologie

- Reines HTML, CSS und JavaScript
- Kein Framework, keine externen Bibliotheken
- Responsives Grid-Layout mit CSS Custom Properties
